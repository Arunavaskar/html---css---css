Opening links in a new window

target

If you want to open the link to a new window, you can use this target="_blank" after adding the link in a quote as a value to the href attribute in the < a > tag.
One of the major reasons a web page author might want a link to be opened is simply because
opening a link on the same page might distract the user from the main page and opening in a new page
helps the user to comeback to the main page easily. Generally avoid opening a link in a new window but
if you do, try to inform the user that link will be opened in a new window after being clicked.
