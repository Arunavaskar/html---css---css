What is a table?

A table represents information in a grid format.
Example of a table Includes TV schedules, sports results, financial reports.

Grids allows us to understand complex data
by referencing information on two axes.

Each block in a grid is referred to as table cells.
In HTML a table is written row by row.

Basic Table Structure

"< table >"

The Table element is used create a table.
The content of a table is written row by row.

"< tr >"

You can indicate the starting of a table row by
using this opening tag(tr stands for table row).

It is followed by one or more < td > elements(one for
each cell in that row).

At the end of the row you use a closing tag.

"< td >"

Each cell of a table is represented using td element(td stands for
data).

at the end of each cell you use a closing tag.

Table Headings

< th >

To create a table heading, you need to use this element.
This element is just like td element just that it's purpose is to represent
heading for a column or a row. th stands for table headings.

Even if a cell has no content, you sould still use a < td > or a
< th > element to represent the presence of a empty cell other wise
that table won't be rendered correctly.

You can use the scope attribute to the < th > element to indicate
if the heading is a column or a row. This attribute can take
row to indicate if the heading is for a row or col to indicate if the
heading is for a col.

browsers normaly display content of a th element in bold and in the middle of a cell.
