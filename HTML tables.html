There are several types of information
that need to be displayed in a grid or table. For example:
Stock results, Stock reports, Train time tables.

When representing information in a table,
you need to think of a grid made up of rows and columns
(kind of like spreadsheet). We learn to...

  • Use four key elements to create Tables
  • Represent complex data using tables
  • Adding captions to tables

What is a table?

A table represents information in a grid format.
Example of a table Includes TV schedules, sports results, financial reports.

Grids allows us to understand complex data
by referencing information on two axes.

Each block in a grid is referred to as table cells.
In HTML a table is written row by row.
