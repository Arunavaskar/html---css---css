Bold & Italics!

"< b >"

This element is a old element for html used to
bold out a selected portion of a sentence.
There is a newer "< strong >" element used to do the samne thing.


"< i >"

This element is also a old element of html
used to make any selected portion of a sentence italic.
There is a "< em >" element used to do the same thing.

Superscript & Subscript

"< sup >"

This element is used to contain charecters that are superscripts, such as the suffix of dates or
mathematical concepts like raising a number to a power like "32".


"< sub >"

This element is used to contain charecters that are Subscripts,
is commonly used for foot notes and chemical formulas. Like "Co2".

Strong & Emphasis!

"< strong >"

This element is kind of a new version of bold element and is
used to Bold out a selected portion of a sentence.

"< em >"

This element stands for emphasis and is used to emphasise on
a particular sentence. By default browsers display it's contains in itaics.

This is used for longer quotes that take up an entire

And you can change it, you can influence it… Once you learn that, you’ll never be the same again.” ― Steve Jobs

"< q >"

This is used for shorter quotes.

Don’t compare yourself with anyone in this world…if you do so, you are insulting yourself. - Bill Gates

Citation & Defination

"< cite >"

This element is used to indicate where the citation is from,
when you are referencing a book or a film or a paper. This element should not be
used for any person's name. Browsers will render this element's contents in italic.

like:- "This is a book that i always like to read; Relativity: The Special and the General Theory by Albert Einstein "

"< dfn >"

This element stands for Defination and is used to explain some kind of new terminology,
academic concepts in a document. It is a defining instance of it.
This element is used to indicate the defining instance of a new term.
some browsers display this element's contents in italic but safari and chrome dosn't change anything!.

Like:- "A Warm Hole is just a fiction based on theories based on fictional theories made by theoritical physisists!"
