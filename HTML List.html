There is a lots of occasions when we need to use lists.
HTML provides us with three different types:

  • Ordered lists are lists where each item in the list is numbered.
    for example the list might be a set of steps that starts and ends with 1 to some number.
    those numbers can be like 1 or romans!
  • Unordered lists begins and ends with bullets rather than charecters that indicate oreder!
  • Definition lists are made up of terms along with there definitions following them.

Ordered List

"< li >"

Each item in the list is placed between an opening "< li >" aand a closing "" tag.(li stands for list items.)
Browsers indent lists by default.


"< ol >"

The Ordered list is created with this element
Sometimes you will see type of attribute used with the ol element to specify the type numbering like
numbers, letters or romans. It is better to use CSS list-style-type property covered on page SOMETHING.

Like:-

  1. oredered list 1
  2. ordered list 2
  3. oredered list 3


"< ul >"

This element is used to create unordered list.
Sometimes you will see type of attribute used with the ol element to specify the type numbering like
numbers, letters or romans. It is better to use CSS list-style-type property covered on page SOMETHING.

Like:-

  • unoredered list 1
  • unordered list 2
  • unoredered list 3

Defininition List

"< dl >"

The definition list is created using this element and is usually
used to create list consisting of terms following there definitions.
Inside dl element you will find dt and dd element.


"< dt >"

This element is used to contain the term being described(the definition term)


"< dd >"

This element is used to contain the definition.

Sometimes you might see two terms containing one or more definitions or
one term containing more than one definition.

Like:-

Term 1

Term 2

more than one definition to one term!

Nested List

You can nest another list inside a list. This called nested list.
Browsers ususally indent the nested list more than a normal list and
for unordered lists browsers may change the bullet type to something different too.

Like:-

  1. ordered parent List no 1
  2. oredered parent List no 2
